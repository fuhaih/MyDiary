# 什么是Blazor？
微软的webassembly

# VS Code中运行第一个Blazor项目
* 安装较新版本的.NET SDK（.NET Core SDK）
* 安装Blazor模板

        dotnet new -i Microsoft.AspNetCore.Blazor.Templates
* 创建Blazor项目

        dotnet new blazor -o BlazorApp1//BlazorApp1项目名
* 终端转换到项目路径下（手动vs code选择也行）
        
        cd BlazorApp1

* 运行项目
        
        dotnet run
* 浏览项目
