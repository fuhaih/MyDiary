## 环境配置(centos7)
* 安装nodejs

```vim shell
$ curl --silent --location https://rpm.nodesource.com/setup_8.x | bash -
$ yum install -y nodejs
```

* 安装git
```vim shell
$ yum -y install git
//查看版本号
$ git --version
```
* 通过node的包管理工具npm安装hexo

```vim shell
$ npm install -g hexo-cli
```
## 建站
```vim shell
$ hexo init <folder>
$ cd <folder>
$ npm install
```
